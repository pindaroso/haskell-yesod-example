# haskell-yesod-example

**Haskell Yesod Application Example with Stack and Nix Builds**

## Requirements

* Vagrant and Ansible or Docker and Docker Compose

## Setup

```bash
vagrant up
vagrant ssh -c "cd code && make"
```
